# Clock Gated Carry Increment Adder

## Overview
This project presents the design and implementation of a Carry Increment Adder (CIA) integrated with conventional Clock Gating techniques for low-power VLSI applications. The objective is to reduce unnecessary switching activity and dynamic power consumption by controlling clock propagation to inactive circuit blocks.

The design is developed using Verilog HDL and analyzed using Cadence EDA tools to evaluate performance metrics such as power, area, and delay.

## Objectives
- Design a Carry Increment Adder (CIA).
- Implement conventional Clock Gating techniques.
- Reduce dynamic power consumption.
- Analyze area, power, and timing performance.
- Verify functionality through simulation waveforms.

## Tools Used
- Cadence Virtuoso
- Cadence Genus
- Cadence Xcelium / Incisive
- Verilog HDL

## Methodology
1. Design the Carry Increment Adder architecture.
2. Integrate Clock Gating logic.
3. Develop RTL code using Verilog HDL.
4. Perform functional simulation.
5. Verify output through waveform analysis.
6. Synthesize the design using Cadence tools.
7. Evaluate:
   - Area
   - Power
   - Delay
